## Prerequisites

Make sure you have the required CLIs installed:

- **Docker** — `docker` and `docker-compose`
- **Vercel** — `npm i -g vercel`
- **Railway** — `npm i -g @railway/cli`
- **Fly.io** — Install from [fly.io](https://fly.io)

---

## Authentication

Cloud platforms require authentication:

```bash
# Vercel
vercel login

# Railway
railway login

# Fly.io
fly auth login
```

---

## Best Practices

1. **Test locally** before deploying
2. **Use environment variables** for secrets
3. **Review deployment config** before confirming
4. **Check logs** after deployment
