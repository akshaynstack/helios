## Prerequisites

Playwright is included with Helios. On first use, it may install browser binaries:

```bash
npx playwright install chromium
```

---

## Headless Mode

By default, browser runs in headless mode (no visible window).

For debugging, you can request visible mode:

```
open the browser visibly and go to example.com
```

---

## Selectors

Use CSS selectors or text content:

- **CSS:** `#login-button`, `.submit-btn`
- **Text:** `button:has-text("Submit")`
- **ID:** `#email-input`
