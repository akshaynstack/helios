## üîß AutoFix (Self-Healing Mode)

The most powerful tool in the Helios arsenal. Ask Helios to auto-fix your project: 

```
auto-fix my project at http://localhost:3000
```

### How It Works
1. **Audit**: Scans the page for SEO, accessibility, and design issues.
2. **Fix Plan**: Generates a prioritized list of issues with specific fix suggestions.
3. **Loop with Guards**: Tracks iteration count to prevent infinite loops.
4. **Stop Conditions**: Automatically stops when:
   - The target **Vibe Score** is reached (default: 85).
   - **Max iterations** are reached (default: 3).
   - A **HARD_STOP** is triggered.

### Credit Protection üí∞
This tool is designed to **protect your API credits**. The AI is instructed to stop when the status is `optimization_complete` or `max_iterations_reached`. It will not loop forever.
