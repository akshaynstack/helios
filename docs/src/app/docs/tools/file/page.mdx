## Permissions

File operations require permission before execution:

- **Read operations** — Usually auto-approved
- **Write/Edit operations** — Requires confirmation
- **Delete operations** — Always requires confirmation

Use `/yolo` mode to skip permission prompts.

---

## Best Practices

1. **Be specific** — Mention exact file paths when possible
2. **Review changes** — Check the diff before approving edits
3. **Use relative paths** — Helios works from your current directory
