## Security

Shell commands are security-scanned before execution:

**Blocked patterns:**
- `rm -rf /` and destructive variations
- `eval()` and code injection
- Suspicious file operations

**Requires confirmation:**
- All shell commands (unless in YOLO mode)
- System-modifying operations

---

## Cross-Platform

Commands are executed using the system's default shell:

- **Windows** — PowerShell
- **macOS/Linux** — bash/zsh

---

## Best Practices

1. **Review commands** before approving
2. **Use specific commands** instead of scripts when possible
3. **Avoid sudo** unless necessary
4. **Check output** for errors
