## Setting API Keys

Use the `config set` command:

```bash
helios config set <KEY_NAME> <value>
```

### OpenRouter (Recommended)

OpenRouter provides access to many models, including free ones:

```bash
helios config set OPENROUTER_API_KEY sk-or-v1-xxxx
```

Get your key: [openrouter.ai/keys](https://openrouter.ai/keys)

### Anthropic

For direct Claude API access:

```bash
helios config set ANTHROPIC_API_KEY sk-ant-xxxx
```

Get your key: [console.anthropic.com](https://console.anthropic.com)

### OpenAI

For GPT models:

```bash
helios config set OPENAI_API_KEY sk-xxxx
```

Get your key: [platform.openai.com](https://platform.openai.com)

### Google

For Gemini models:

```bash
helios config set GOOGLE_API_KEY AIzaxxxx
```

Get your key: [aistudio.google.com](https://aistudio.google.com)

## Viewing Configuration

See all configured keys:

```bash
helios config show
```

Output:

```
⚙️  Helios Configuration

● OPENROUTER_API_KEY
  OpenRouter API: sk-or-...xxx
○ OPENAI_API_KEY
  OpenAI API: (not set)
● ANTHROPIC_API_KEY
  Anthropic API: sk-ant-...xxx
```

---

## Priority Order

When multiple keys are configured, Helios uses this priority:

1. **Anthropic** (highest priority)
2. **OpenRouter**
3. **OpenAI**
4. **Google**

This means if you have both Anthropic and OpenRouter configured, Helios will use Anthropic by default.

## V0 API Key (Optional)

For UI generation features:

```bash
helios config set V0_API_KEY v0-xxxx
```

Get your key: [v0.dev/chat/settings/keys](https://v0.dev/chat/settings/keys)

---

## Environment Variables

You can also use environment variables:

```bash
export OPENROUTER_API_KEY=sk-or-v1-xxxx
export ANTHROPIC_API_KEY=sk-ant-xxxx
```

> Note: Config file settings take precedence over environment variables.

---

## Troubleshooting

### Key Not Working

Run the doctor command:

```bash
helios doctor
```

This will test your API connections and report issues.

### Rate Limits

If you're hitting rate limits:

1. Switch to a different provider
2. Use a model with higher limits
3. Upgrade your API plan
