# Models Configuration

Configure AI models and providers for Helios.

---

## Changing Models

Use the `/model` command in chat mode:

```
/model
```

This will:
1. Show available providers (based on your configured API keys)
2. Fetch models from the selected provider
3. Let you search and select a model

## Setting Up Providers

### OpenRouter (Recommended)

Access 100+ models including free options:

```bash
helios config set OPENROUTER_API_KEY sk-or-v1-...
```

Get a key at [openrouter.ai/keys](https://openrouter.ai/keys)

### OpenAI

Direct access to GPT models:

```bash
helios config set OPENAI_API_KEY sk-...
```

### Anthropic

Claude models via direct API:

```bash
helios config set ANTHROPIC_API_KEY sk-ant-...
```

### Google

Gemini models:

```bash
helios config set GOOGLE_API_KEY ...
```

## Custom Providers

Add any OpenAI-compatible endpoint (Ollama, LM Studio, vLLM, etc.):

```bash
# Add a custom provider
helios provider add http://localhost:11434/v1

# With API key (if required)
helios provider add https://my-server.com/v1 my-api-key

# Remove custom provider
helios provider remove

# Show all providers
helios provider show
```

### Common Custom Providers

**Ollama:**
```bash
helios provider add http://localhost:11434/v1
```

**LM Studio:**
```bash
helios provider add http://localhost:1234/v1
```

**vLLM:**
```bash
helios provider add http://localhost:8000/v1
```

## Default Model

Set a default model to use:

```bash
helios config set MODEL google/gemini-2.0-flash-exp:free
```

Or use `/model` in chat mode to select interactively.
