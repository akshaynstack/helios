## Setting Values

```bash
helios config set <KEY> <value>
```

### Examples

**Set timeout to 2 minutes:**
```bash
helios config set TIMEOUT 120000
```

**Enable streaming:**
```bash
helios config set STREAMING true
```

**Enable compact mode:**
```bash
helios config set COMPACT_MODE true
```

---

## View Current Settings

```bash
helios config show
```

---

## Streaming Mode

When enabled (default), AI responses stream in real-time.

Toggle in chat:
```
/stream
```

Or via config:
```bash
helios config set STREAMING false
```

---

## Compact Mode

Reduces output verbosity for a cleaner experience.

Toggle in chat:
```
/compact
```

Or via config:
```bash
helios config set COMPACT_MODE true
```

---

## Timeout Configuration

Default is 60 seconds (60000ms). Increase for complex tasks:

```bash
helios config set TIMEOUT 120000
```

---

## Config File Location

Settings are stored at:

- **macOS/Linux:** `~/.config/helios-cli/config.json`
- **Windows:** `%APPDATA%\helios-cli\config.json`
