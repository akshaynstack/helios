# Introduction

Welcome to **Helios Engine** ‚Äî the flow-state AI power tool built for **Vibecoders**. 

Helios isn't just another chat interface; it's a high-performance orchestration engine designed to stay out of your way while providing instant, elite-level context through a suite of Rust-powered tools and proactive automation.

---

## The Vibecoder Philosophy

Helios is designed for developers who value **flow, speed, and intuition**. It eliminates the common friction points of AI assistance:

- **Ultra-Fast Context**: Integrated `ripgrep`, `fd`, and `mgrep` for near-instant search and discovery.
- **Zero Interruption**: Proactive installers set up your environment silently in the background.
- **Transparent Power**: Real-time analytics proving exactly how much manual effort is automated.
- **Unstoppable Logic**: Smart fallbacks for tool-calls and multi-provider model mapping.

## Quick Start

```bash
npm install -g helios-cli
helios config set OPENROUTER_API_KEY sk-or-...
helios
```

That's it! You're ready to start coding with AI assistance.
