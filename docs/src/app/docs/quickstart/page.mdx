# Quick Start

Get productive with Helios in under 5 minutes.

---

## Start Interactive Mode

The simplest way to use Helios is interactive mode:

```bash
helios
```

You'll see the Helios banner and a prompt waiting for your input.

---

## Your First Conversation

Try asking Helios to analyze your project:

```
? How can I help you today? analyze this project and tell me what it does
```

Helios will:
1. Read your project files
2. Understand the structure
3. Provide a detailed summary

## Single Command Mode

For quick tasks, pass your request directly:

```bash
helios "create a README.md for this project"
```

```bash
helios "fix the TypeScript errors in src/"
```

```bash
helios "add error handling to the API routes"
```

---

## UI Generation

Generate beautiful UI components with Vercel v0:

```bash
# First, set your v0 API key
helios config set V0_API_KEY v0-...

# Then generate UI
helios ui "modern dashboard with dark theme"
```

---

## File Operations

Helios can read, write, and edit files automatically:

**Read a file:**
```
show me the contents of package.json
```

**Create a file:**
```
create a new file called utils.ts with helper functions for string manipulation
```

**Edit a file:**
```
add error handling to the fetchData function in api.ts
```

---

## Git Integration

Work with Git directly:

```
what changes have I made?
```

```
commit all my changes with a descriptive message
```

```
show me the git log for the last 5 commits
```

## Pro Tips

### Auto-Approve Mode

Skip permission prompts with YOLO mode:

```
/yolo
```

> ⚠️ Use with caution — this auto-approves all tool executions.

### Compact Mode

Reduce output verbosity:

```
/compact
```

### Change Model Mid-Session

```
/model
```

Select from available models in your configured provider.
